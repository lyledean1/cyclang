# IntelliJ IDEA - Cyclang Syntax Highlighting - Quick Start

## ⚡ 3-Step Setup

### Step 1: Install TextMate Bundles Plugin

1. Open IntelliJ IDEA
2. `Cmd+,` (Settings)
3. **Plugins** → **Marketplace**
4. Search: **TextMate Bundles**
5. Click **Install**
6. **Restart** IntelliJ

### Step 2: Add Cyclang Bundle

1. `Cmd+,` (Settings)
2. **Editor** → **TextMate Bundles**
3. Click **+** button
4. Navigate to: `../cyclang-intellij-bundle/Cyclang.tmbundle`
5. Click **OK**
6. Click **Apply**
